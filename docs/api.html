APIÔÉÅ

This API allows interacting with a collection of frictionless datapackages, stored in a specific format, explore the content of its entries and visualize the underlying data.

A Collection of datapackages, denoted as entries can be created from local files or a remote repository. The metadata describing the data are stored as descriptors and can be accessed via metadata objects. For collections containing a certain type of data, collections with specific methods can be created, such as with Echemdb. In a same way, specific types of entries can be created, such as with EchemdbEntry
